```
"こんにちは世界"という文字列を表示するプログラムを作成してください。
```


## 演習問題 1-3

```
printf()の書式指定を使用して、実行例のとおり表示してください。
printf()は何度使用しても良いです。
```

`実行例`

```
1
3.140000
A
ABC
あ
C言語
```

`模範回答`
<details>
<summary>回答を見る</summary>

printf()を1回使用した時

```c
#include <stdio.h>

int main() {
    printf("%d\n%f\n%c\n%s\n%s\n%s\n", 1, 3.14, 'A', "ABC", "あ", "C言語");
    return 0;
}
```

printf()を複数回使用した時

```c
#include <stdio.h>

int main() {
    printf("%d\n", 1);
    printf("%f\n", 3.14);
    printf("%c\n", 'A');
    printf("%s\n", "ABC");
    printf("%s\n", "あ");
    printf("%s\n", "C言語");
    return 0;
}
```
</details>

## 演習問題 1-9

```

```

`実行例`

```

unsigned int b = 20
long c = 30
unsigned long d = 40
short e = 50
unsigned short f = 60

```

<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

    int a = 10;
    unsigned int b = 20;
    long c = 30;
    unsigned long d = 40;
    short e = 50;
    unsigned short f = 60;
    char g = 70;
    unsigned char h = 80;

    printf("int a = %d\n", a);
    printf("unsigned int b = %d\n", b);
    printf("long c = %ld\n", c);
    printf("unsigned long d = %ld\n", d);
    printf("short e = %d\n", e);
    printf("unsigned short f = %d\n", f);
    printf("char g = %d\n", g);
    printf("unsigned char h = %d\n", h);

    return 0;
}
```

## 演習問題 1-13

```
printf()の書式設定を使用して、実行例のように表示してください。
※実行例の中で、数字で表記されているモノは整数値または実数値として扱ってください。
```

`実行例`

```
  ID 商品名  価格  重量
0001     魚   198  12.3
0002     肉   560 151.5
0003   野菜   224  85.7
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

int main() {
    printf("%4s %7s %7s %7s\n", "ID", "商品名", "価格", "重量");
    printf("%04d %7s %5d %5.1f\n", 1, "魚", 198, 12.3);
    printf("%04d %7s %5d %5.1f\n", 2, "肉", 560, 151.5);
    printf("%04d %8s %5d %5.1f\n", 3, "野菜", 224, 85.7);

    return 0;
}
```
</details>

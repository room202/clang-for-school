## ■ 変数

## 演習問題 1-7

```
1.int型の変数xを宣言してください。
2.int型の変数yを宣言してください。
3.変数xに100を代入してください。
4.変数yに200を代入してください。
5.変数xとyの内容をprintf()で表示してください。
```

`実行結果`

```
変数xの値は100です。
変数yの値は200です。
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

int main() {
    int x;
    int y;
    x = 100;
    y = 200;
    printf("変数xの値は%dです。\n", x);
    printf("変数yの値は%dです。\n", y);
    return 0;
}
```

## ■ 数値型

## 演習問題 1-8

```
1.int型の変数xを宣言してください。
2.int型の変数yを宣言してください。
3.変数xに10を代入してください。
4.変数yに20を代入してください。
5.変数yに変数xを代入してください。
6.変数xとyの内容をprintf()で表示してください。
```

`実行結果`

```
変数xの値は10です。
変数yの値は10です。
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

main()
{
    int x;
    int y;
    x = 10;
    y = 20;
    y = x;
    printf("変数xの値は%dです。\n", x);
    printf("変数yの値は%dです。\n", y);
}
```

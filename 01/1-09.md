## ■ 数値型

## 演習問題 1-9

```
整数型の変数を全て使用して、表示してください。
```

`実行結果`

```
int a = 10
unsigned int b = 20
long c = 30
unsigned long d = 40
short e = 50
unsigned short f = 60
char g = 70
unsigned char h = 80
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

main()
{
    int a = 10;
    unsigned int b = 20;
    long c = 30;
    unsigned long d = 40;
    short e = 50;
    unsigned short f = 60;
    char g = 70;
    unsigned char h = 80;

    printf("int a = %d\n", a);
    printf("unsigned int b = %d\n", b);
    printf("long c = %ld\n", c);
    printf("unsigned long d = %ld\n", d);
    printf("short e = %d\n", e);
    printf("unsigned short f = %d\n", f);
    printf("char g = %d\n", g);
    printf("unsigned char h = %d\n", h);
}
```

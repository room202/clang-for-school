## ■ 数値型

## 演習問題 1-10

```
実数型の変数を全て使用して、表示してください。
```

`実行結果`

```
float i = 3.140000
double j = 42.195000
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

int main() {
    float i = 3.14;
    double j = 42.195;

    printf("float i = %f\n", i);
    printf("double j = %f\n", j);

    return 0;
}
```
</details>

## ■ 文字型

## 演習問題 1-11

```
1.char型の変数cに、文字'R'を代入してください。
2.変数cの中身の文字と文字コードを表示してください。
```

`実行結果`

```
文字 = R
文字コード = 82
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

int main() {
    char c = 'R';

    printf("文字 = %c\n", c);
    printf("文字コード = %d\n", c);

    return 0;
}
```
</details>

## ■ 文字列

## 演習問題 1-12

```
1.char型の配列str[10]を定義してください。
2.配列strに、strcpy()を使用して"こんにちは"を代入してください。
3.printf()で、strの内容を表示してください。
```

`実行結果`

```
こんにちは
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>
#include <string.h>

main()
{
    char str[10];
    strcpy(str, "こんにちは");
    printf("%s\n", str);
}
```


## ■ printf()と定数

## 演習問題 1-5

```
15から37を引いた計算結果を、printf()で表示してください。
数値はすべて "書式指定" を使用してください。
```

`実行結果`

```
15から37を引いた値は-22です。
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

int main() {
    printf("%dから%dを引いた値は%dです。\n", 15, 37, 15-37);
    return 0;
}
```
</details>

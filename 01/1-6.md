## ■ 変数

## 演習問題 1-6

```
1.int型の変数xを宣言してください。
2.その変数xに15を代入してください。
3.最後に、変数xの内容をprintf()で表示してください。
```

`実行結果`

```
変数xの値は15です。
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

int main() {
    int x;
    x = 15;
    printf("変数xの値は%dです。\n", x);
    return 0;
}
```
</details>

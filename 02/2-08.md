## ■ 論理演算子

## 演習問題 2-8

```
1. scanf()を使用して、整数値を１つ入力させてください
2. 入力された整数値が、「100ではない」場合は、
   値として1が、それ以外の場合は、0が代入されている事を確認してください。
```

`実行結果`

```
入力してください : 0
1

入力してください : 100
0

入力してください : 50
1

入力してください : -1
1
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

main()
{
    int input = 0;
    int n = 0;

    printf("入力してください : ");
    scanf("%d", &input);

    n = !(input == 100);
    printf("%d\n", n);
}
```

## ■ 型の変換

## 演習問題 2-12

```
1. scanf()を使用して、整数値を１つ入力させてください。
2. 入力された整数値を「3.0」で割って、小数点以下第3位まで表示してください。
```

`実行結果`

```
入力してください > 9
3.000

入力してください > 10
3.333
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

int main() {

    int input = 0;

    printf("入力してください > ");
    scanf("%d", &input);

    printf("%.3f\n", input / 3.0);
    
    return 0;
}
```

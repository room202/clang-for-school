## ■ 型の変換

## 演習問題 2-13

```
1. scanf()を使用して、整数値を１つ入力させてください。
2. 入力された整数値をfloat型に型変換(キャスト)して、小数点以下第3位まで表示してください。
```

`実行結果`

```
入力してください > 9
9.000

入力してください > 10
10.000
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

int main() {

    int input = 0;

    printf("入力してください > ");
    scanf("%d", &input);

    printf("%.3f\n", (float)input);
    
    return 0;
}
```
</details>

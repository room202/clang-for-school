## ■ 計算の演算子（２）

## 演習問題 2-3

```
整数型の変数xの初期値が5で、xをインクリメントして3回表示するプログラムを作成してください。
```

`実行結果`

```
5
6
7
8
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

int main() {
    int x = 5;
    printf("%d\n", x);
    x++;
    printf("%d\n", x);
    x++;
    printf("%d\n", x);
    x++;
    printf("%d\n", x);
    return 0;
}
```
</details>

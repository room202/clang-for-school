## ■ 計算の演算子（２）

## 演習問題 2-4

```
整数型の変数yの初期値が10で、yをデクリメントして4回表示するプログラムを作成してください。
```

`実行結果`

```
10
9
8
7
6
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

int main() {
    int y = 10;
    printf("%d\n", y);
    y--;
    printf("%d\n", y);
    y--;
    printf("%d\n", y);
    y--;
    printf("%d\n", y);
    y--;
    printf("%d\n", y);
    return 0;
}
```
</details>

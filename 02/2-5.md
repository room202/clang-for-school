## ■ 比較演算子

## 演習問題 2-5

```
下記比較演算子を全て使用して、条件成立時は「1」、条件不成立時は「0」となる事を確認してください。
==
!=
<
>
<=
>=
```

`実行結果`

```
11 == 11 : 1
11 == 12 : 0
11 != 11 : 0
11 != 12 : 1
11 <  12 : 1
12 <  11 : 0
12 >  11 : 1
11 >  12 : 0
11 <= 12 : 1
12 <= 11 : 0
12 <= 12 : 1
12 >= 11 : 1
11 >= 12 : 0
11 >= 11 : 1
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

int main() {
    int n = 0;
    
    n = 11 == 11;
    printf("11 == 11 : %d\n", n);

    n = 11 == 12;
    printf("11 == 12 : %d\n", n);

    n = 11 != 11;
    printf("11 != 11 : %d\n", n);

    n = 11 != 12;
    printf("11 != 12 : %d\n", n);

    n = 11 < 12;
    printf("11 <  12 : %d\n", n);

    n = 12 < 11;
    printf("12 <  11 : %d\n", n);

    n = 12 > 11;
    printf("12 >  11 : %d\n", n);

    n = 11 > 12;
    printf("11 >  12 : %d\n", n);

    n = 11 <= 12;
    printf("11 <= 12 : %d\n", n);

    n = 12 <= 11;
    printf("12 <= 11 : %d\n", n);

    n = 12 <= 12;
    printf("12 <= 12 : %d\n", n);

    n = 12 >= 11;
    printf("12 >= 11 : %d\n", n);

    n = 11 > 12;
    printf("11 >= 12 : %d\n", n);

    n = 12 >= 12;
    printf("12 >= 12 : %d\n", n);

    return 0;
}
```

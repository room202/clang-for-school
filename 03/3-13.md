## ■ switch文

## 演習問題 3-13

```
switch文を使って
ユーザーに曜日を表す整数値を入力させ、
入力された値に対応する曜日の名前を表示するプログラムを作成してください。
ただし、1から7までの整数値が有効な値とします。
入力された整数値が1から7の範囲外の場合、「無効な値です。」と表示してください。
※1は日曜日、7は土曜日とする。
```

`実行結果`

```
曜日を表す整数値を入力してください（1-7）: 3
入力された値に対応する曜日は「水曜日」です。
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

main()
{
    int day;
    printf("曜日を表す整数値を入力してください（1-7）: ");
    scanf("%d", &day);

    switch (day) {
        case 1:
            printf("入力された値に対応する曜日は「日曜日」です。\n");
            break;
        case 2:
            printf("入力された値に対応する曜日は「月曜日」です。\n");
            break;
        case 3:
            printf("入力された値に対応する曜日は「火曜日」です。\n");
            break;
        case 4:
            printf("入力された値に対応する曜日は「水曜日」です。\n");
            break;
        case 5:
            printf("入力された値に対応する曜日は「木曜日」です。\n");
            break;
        case 6:
            printf("入力された値に対応する曜日は「金曜日」です。\n");
            break;
        case 7:
            printf("入力された値に対応する曜日は「土曜日」です。\n");
            break;
        default:
            printf("無効な値です。\n");
    }
}
```

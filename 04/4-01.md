## ■ 配列

## 演習問題 4-1

```
整数(int型)の1次元配列が与えられます。
配列のすべての要素の合計を計算し、その値を出力するC言語のプログラムを作成してください。
配列の要素数は5とします。
※配列の要素の値は、好きな数値を入れてください。
```

`実行結果`

※配列：{1, 2, 3, 4, 5}の場合

```
合計: 15
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

main()
{
    int numbers[5] = {1, 2, 3, 4, 5};
    int sum = 0;

    for(int i = 0; i < 5; i++) {
        sum += numbers[i];
    }

    printf("合計: %d\n", sum);
}
```

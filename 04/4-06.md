## ■ 文字列自由自在

## 演習問題 4-6

```
① 文字列型配列str1を宣言し、 "World!" で初期化せよ
② 文字列型配列str2を宣言し、str1の文字列をstr2にコピーせよ
③ 文字列型配列str3を宣言し、"Hello" で初期化せよ
④ str3の保持する文字列の後ろに "World!" を結合せよ
⑤ str1とstr2が同じ文字列であれば、
　「同じ文字列です。」と、異なれば
　「異なる文字列です。」と画面に表示せよ
⑥ 画面に、str1, str2, str3の文字列を表示せよ
※①～④の順番は、変わっても構わないです。
```

`実行結果`


```
同じ文字列です。
World!
World!
HelloWorld!
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>
#include <string.h>

main()
{
    // ① 文字列型配列str1を宣言し、 "World!" で初期化せよ
    char str1[] = "World!";

    // ② 文字列型配列str2を宣言し、str1の文字列をstr2にコピーせよ
    char str2[32] = {0};
    strcpy(str2, str1);

    // ③ 文字列型配列str3を宣言し、"Hello" で初期化せよ
    char str3[32] = "Hello";

    // ④ str3の保持する文字列の後ろに "World!" を結合せよ
    strcat(str3, "World!");

    // ⑤ str1とstr2が同じ文字列であれば、
    //   「同じ文字列です。」と、異なれば
    //   「異なる文字列です。」と画面に表示せよ
    if(strcmp(str1, str2) == 0) {
      printf("同じ文字列です。\n");
    }
    else {
      printf("異なる文字列です。\n");
    }

    // ⑥ 画面に、str1, str2, str3の文字列を表示せよ
    printf("%s\n", str1);
    printf("%s\n", str2);
    printf("%s\n", str3);
}
```

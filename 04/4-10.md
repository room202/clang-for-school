## ■ アドレス

## 演習問題 4-10

```
1. char型の配列を要素数3個で宣言せよ
※配列名は好きな名前で良い
2. 宣言した配列の各要素のアドレスを画面に表示せよ
```

`実行結果`

※必ずしもアドレスはこの実行結果と同じにはなりませんので注意してください。

```
c[0]  : 0x7ffedbfb21ed
c[1]  : 0x7ffedbfb21ee
c[2]  : 0x7ffedbfb21ef
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

main()
{
    char c[3];

    printf("c[0]  : %p\n", &c[0]);
    printf("c[1]  : %p\n", &c[1]);
    printf("c[2]  : %p\n", &c[2]);
}
```
</details>

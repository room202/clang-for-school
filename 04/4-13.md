## ■ NULLポインタ

## 演習問題 4-13

```
1. int型のポインタ変数を ptr 宣言し、NULLポインタで初期化してください。
2. 条件文を使用して、ポインタがNULLかどうかを判定し、結果を表示してください。
```

`実行結果`

```
ポインタはNULLです。
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

main()
{
    int* ptr = NULL;

    if (ptr == NULL) {
        printf("ポインタはNULLです。\n");
    } else {
        printf("ポインタはNULLではありません。\n");
    }
}
```

## ■ NULLポインタ

## 演習問題 4-14

```
1. int型のポインタ変数を ptr 宣言し、NULLポインタで初期化してください。
2. int型の変数 i を宣言してください。
3. 条件文を使用して、ポインタがNULLかどうかを判定し、結果を表示してください。
4. その後、ptr に i のアドレスを代入してください。
5. 条件文を使用して、ポインタがNULLかどうかを判定し、結果を表示してください。
※if-else文を２回使用して判定してください。
```

`実行結果`

```
ポインタはNULLです。
ポインタはNULLではありません。
```

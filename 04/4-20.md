## ■ ポインタと配列

## 演習問題 4-20

```
整数型の配列 array が与えられています。
配列内の最大値を見つけ、その最大値とそのインデックスを表示するC言語のプログラムを作成してください。
ただし、最大値とそのインデックスを見つけるためにポインタを使用してください。
```

`ソースコード`

```c
#include <stdio.h>

main()
{
    int array[7] = {1, 3, 5, 7, 9, 2, 4};
    int *p;

    // ヒント: ポインタを使用して配列を順番にスキャンし、それぞれの要素が現在の最大値より大きいかどうかを確認します。

    // ポインタを使用して array の中の最大値とそのインデックスを見つける

    // 以下に最大値とそのインデックスを表示する
    // printf("最大値: %d, インデックス: %d\n", 最大値, インデックス);
    
}
```

`実行結果`

```
最大値: 9, インデックス: 4
```

#include <stdio.h>

main()
{
    int array[7] = {1, 3, 5, 7, 9, 2, 4};
    int *p;

    // ポインタを使用して array の中の最大値とそのインデックスを見つける
    int max = *array; // 初期の最大値は配列の最初の要素とする
    int index = 0; // 初期の最大値のインデックスは0とする

    for (p = array; p < array + 7; p++) {
        if (*p > max) {
            max = *p;
            index = p - array; // pとarrayの差分で最大値の位置（インデックス）を得る
        }
    }

    // 以下に最大値とそのインデックスを表示する
    printf("最大値: %d, インデックス: %d\n", max, index);
}

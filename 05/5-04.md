## ■ 関数の定義

## 演習問題 5-4

```
整数の引数を受け取り、その値を2倍にして表示する関数 doubleValue を作成してください。
その後、関数を呼び出し、結果を表示してください。
```

`実行結果`

```
整数値を入力 : 5
2倍した値: 10
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

void doubleValue(int num) {
    num *= 2;
    printf("2倍した値: %d\n", num);
}

main()
{
    int number = 0;
    printf("整数値を入力 : ");
    scanf("%d", &number);

    doubleValue(number);
}
```
</details>

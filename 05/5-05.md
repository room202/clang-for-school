## ■ 関数の定義

## 演習問題 5-5

```
整数型の引数を2つ受け取り、それらの値の和を計算して返す関数 calculateSum を作成してください。
その後、関数を呼び出し、結果を表示してください。
```

`実行結果`

```
整数値を入力1 : 5
整数値を入力2 : 10
足した値: 15
```

`模範回答`
<details>
<summary>回答を見る</summary>

```c
#include <stdio.h>

int calculateSum(int num1, int num2) {
    int sum = num1 + num2;
    return sum;
}

main()
{
    int a = 0;
    int b = 0;

    printf("整数値を入力1 : ");
    scanf("%d", &a);

    printf("整数値を入力2 : ");
    scanf("%d", &b);

    int result = calculateSum(a, b);

    printf("足した値: %d\n", result);
}
```
